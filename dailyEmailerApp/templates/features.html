Features

Email Scheduling Options

Users can schedule emails to be sent: Daily, Weekly, Monthly, Yearly, and Once on a specific date.

Email Management

Centralized management of all scheduled emails. Ability to edit email content before sending. Options to activate, pause, or deactivate scheduled emails as per user preference.

Toxicity Filtering

Real-time toxicity check during email composition. Notifications for inappropriate content like sexual references, profanity, insults, threats, and identity attacks.

HTML Email Support

Creation and sending of HTML-formatted emails. Enables rich text, images, and formatting options for enhanced email content.

Custom Mail Server Configuration

Option for users to set up and use their own custom mail server for sending emails.

Default Mail Server Integration

Utilizes a default mail server if users do not configure a custom mail server. Ensures reliable email delivery even without custom settings.

User Interface and Accessibility

Intuitive interface for easy email composition, scheduling, and management. Accessibility features to accommodate diverse user needs.

Delivery Confirmation and Tracking

Provides delivery confirmation and tracking for sent emails. Users can track whether emails were successfully delivered.
